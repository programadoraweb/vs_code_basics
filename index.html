VS Code Basics
Command Function
code . open vs code
code -version version
(select word)+(cntrl + D)+(left/right arrows) select next occurency
alt + click at different places write the samething at different places
cntrl + B open close sidebar
cntrl + W closes file/window
cntrl + N create a new file/window
cntrl + ~ open / closes terminal
cntrl + right or left arrows goes to the end/beggining of the word
shift + right or left arrows selects the same letter with several occurencies
alt + up or down arrows moves sellected text up or down
cntrl + right or left brackets or (tab & shift/tab) indentation
cntrl + P (name of the file or part of the name of the file) find a file
cntrl + P > format document format document with prettier extension
cntrl + c / cntrl + v (with NO selection) will copy and paste the whole line
cntrl + F regular search
cntrl + shift + F search inside all files
F12 shows mozzila s element or attribute definition
& + any letter search for character entity code and visualize it
